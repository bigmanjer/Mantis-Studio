# Cleanup Report

## Full tree summary (top-level folders + purpose)
- `app/`: Shared UI utilities, layout helpers, and navigation bridge into the `mantis` router.
- `assets/`: Brand imagery used by the Streamlit UI.
- `docs/`: Internal documentation and runbooks.
- `legal/`: Legal policy markdown content rendered in-app.
- `mantis/`: Core application modules, router, UI pages, services, and storage logic.
- `pages/`: Streamlit multipage entry points for Account Settings and Legal Center.
- `scripts/`: Developer tooling (healthcheck + smoke tests).
- `.github/`: CI/workflow automation.
- `.streamlit/`: Streamlit configuration and secrets templates.

## Runtime path inventory
- **Entrypoint**: `Mantis_Studio.py` (required by Streamlit Cloud).
- **Routing & navigation**: `mantis/router.py`, `mantis/ui/pages/*`, and `app/utils/navigation.py` (bridge).
- **Data persistence**: `Mantis_Studio.py` project storage logic (writes to `projects/` and `.backups`).
- **Auth**: `app/utils/auth.py` and `mantis/services/auth.py`.
- **Legal & footer**: `legal/*.md`, `pages/Legal Center.py`, and `app/ui/layout.py`.
- **Assets**: `assets/mantis_logo_trans.png`, `assets/mantis_wordmark.png`.
